In this example (to obtain the person name) we are demonstrating the following:

  1. Asynchronously initiating a separate section from a sibling section (separate js file); requireSection(...)
  2. Accessing section exports from a sibling section; sectionExports
  3. Calling intra-section modules (all intra-section module files are packaged intoto a single js file using browserify); require(...)
See sections/base/main.js for the application entry point
