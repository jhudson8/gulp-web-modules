This is your gulp-web-modules jumpstart application.


The following direcories were created
  • public: any files put here will be copied to the root of the deployable contents
  • sections/base: The required "section" for any application. All javascript files within will be "browserified" and merged into a single deployable javascript file
    • index.js: any exports from this module will be available when the "base" section (this section) is required
    • normalize.css: A normalize css file mostly used to demonstrate css file concatination
    • style.css: The other stylesheet which gets merged to {root}/sections/base.css

More details and plugins can be seen in the simple example app.

Some available plugins are:
  • gwm-config: copy build type specific config JSON to the base javascript file
  • gwm-lib: copy all file content in a specific directory to gulp-web-modules base javascript file
  • gwm-react: compile all jsx into javascript
  • gwm-handlebars: precompile all handlebars templates
  • gwm-stylus: compile all stylus files into css
  • gwm-sass: compile all sass/scss files into css
